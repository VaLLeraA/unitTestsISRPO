# Документация проекта

## Описание
Данная программа содержит две функции: subtraction и multiplication и класс TriangleTestCase, который реализует тестирование этих функций с использованием библиотеки unittest

### Функция 1: 'subtraction(a, b)'
Выполняет вычитание числа `b` из числа `a`

#### Аргументы:
* 'a' (int): число, из которого вычитают
* 'b' (int): число, которое вычитают

#### Пример вызова:
'''python
print(subtraction(5, 3)) #2

### Функция 2: 'multiplication(a, b)'
Выполняет умножение числа `a' на число `b`

#### Аргументы:
* 'a' (int): число, которое умножают
* 'b' (int): число, на которое умножают

#### Пример вызова:
'''python
print(multiplication(6, 2)) #12

## Класс: MathTestClass 
Содержит тесты для проверки функциональности функций subtraction и multiplication. Тестирование осуществляется с помощью библиотеки unittest. 

### Тест 1: 'test_multiplication()'
Проверяет корректность работы функции `multiplication` с различными набором входных данных.

#### Тестируемые случаи**:
    `multiplication(1, 2)` -> ожидается `2`
    `multiplication(2, -3)` -> ожидается `-6`
    `multiplication(-5, -5)` -> ожидается `25`
    `multiplication(3, 0)` -> ожидается `0`

#### Запуск тестов:
Чтобы запустить тесты, выполните следующую команду в терминале: python -m unittest имя_вашего_файла

### Тест 2: `test_subtraction()`
Проверяет корректность работы функции `subtraction ` с различными набором входных данных.

#### Тестируемые случаи**:
    `subtraction(1, 2)` -> ожидается `-1`
    `subtraction(2, -3)` -> ожидается `5`
    `subtraction(-5, -5)` -> ожидается `0`
    `subtraction(-5, 2)` -> ожидается `-7`
    `subtraction(3, 0)` -> ожидается `3`

#### Запуск тестов:
Чтобы запустить тесты, выполните следующую команду в терминале: python -m unittest имя_вашего_файла
